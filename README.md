#   二叉树
* 前序遍历
    利用栈、先加右子树、再加左子树
* 中序遍历
    利用栈、遍历左,为空时出栈,打印并将右子树入栈
* 后续遍历
    利用栈、需要记录当前节点和前驱节点。先左后右、一直入栈
    当前节点不存在左右子树、并且前驱节点不为当前节点的左右子树时 
    打印当前节点、出栈
